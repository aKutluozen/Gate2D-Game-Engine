Gate2D is an HTML5 Canvas based game engine written in JavaScript. It provides developers with a useful set of tools for graphics, level management, object pooling, math and physics calculations, and many more. Gate2D helps set up a project structure and build playable games in matter of minutes.

Basic Workflow of the Engine:

  • Custom game objects must be created in assets/objects folder as .js files.
  • All the custom assets (objects, images, sounds, scripts, etc.) must be declared in the setup.js file.
  • Once they are declared, fine tuning of the engine must be also done in setup.js using loadAll function.
  • Global variables and level configurations should be done in entities.js.
  • Any extra draw or update calls can be added to update.js or draw.js.
